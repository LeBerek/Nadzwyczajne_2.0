Exceptional Transport Application

Before exceptional transport takes place and once the consignment is qualified as exceptional consignment the consignee sends to railway undertaking an order for an exceptional consignment permit with conditions of carriage. The railway undertaking obtains a permit with transport conditions from the infrastructure manager.
In international transports the permit is shared with all other participating railway undertakings.

Communication between these entities is carried out in a standardized form, which derives from the rules set in the UIC document IRS 50502 "Exceptional Consignments - Regulations concerning the preparation and management of conceptional consignments". These rules are binding for all UIC railway undertakings involved in the international transports of exceptional consignments.

Appendix B of the UIC IRS 50502 document contains a template of a form of written communication with 38 points. These points can be grouped into the main three elements of the exceptional transports:

  • goods data - which are to be transported,
  • wagon data - which is suitable for the transport of the good,
  • path data - which is ordered by the consignee, and for which infrastructure manager sets and approves conditions of carriage.

Infrastructure manager examines the carriage conditions for the set of the wagon with cargo on the paths within his railways network. Practice shows that in the application forms usually points 1-32 are filled by railway undertakings. In reply, the infrastructure manager sends the permit with some of these 32 points. In Appendix B of the UIC IRS 50502 there is guidance which points are obligatory for each entity in the approval process.

International written communication is conducted in one of the three official UIC official languages: German, French or English. In urgent cases it is accepted to send an approval applications in other non-official language. In the countries where the official UIC languages are not national languages, it is the responsibility of the railway undertaking to translate the content of the application into national language.

This project describes each one of the free application elements with focus on the application codes which describe goods, wagon and path data.
